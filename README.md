# Telegram-бот управления ювелирным ассортиментом

Полнофункциональный Telegram-бот для управления ассортиментом ювелирных изделий через Google Sheets.

## Функциональность

- 🔍 Поиск по категории (Браслеты, Подвески, Серьги, Кольца, Наборы и пользовательские листы)
- 🔎 Поиск по артикулу
- 📝 Управление листами (создание, удаление, просмотр)
- 👁️ Просмотр карточек товаров с фото, названием, артикулом и остатками
- 🔄 Изменение количества товаров (+/-)
- 🗑️ Удаление товаров
- 🔄 Синхронизация с Google Sheets

## Структура Google Таблицы

### Стандартные листы:

#### Браслеты:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул (уникальный идентификатор)
- Колонка D: Количество для размера 16 (13-14см)
- Колонка E: Количество для размера 18 (15-17см)
- Колонка F: Количество для размера 20

#### Подвески, Серьги, Наборы:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул
- Колонка D: Количество (единый размер)

#### Кольца:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул
- Колонка D: Количество для размера 50
- Колонка E: Количество для размера 52
- Колонка F: Количество для размера 54

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd pandorashop
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте переменные окружения в файле `.env`:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
ADMIN_IDS=comma_separated_list_of_admin_telegram_ids
GOOGLE_SHEET_ID=your_google_sheet_id_here
GOOGLE_CREDENTIALS_FILE=credentials.json
```

4. Добавьте файл `credentials.json` с ключами доступа к Google API

5. Запустите бота:
```bash
python bot.py
```

## Использование

- `/start` - начать работу с ботом
- `/search_article` - поиск товара по артикулу
- `/create_sheet` - создание нового листа
- `/sync` - синхронизация с Google Sheets

## Требования

- Python 3.8+
- Аккаунт Telegram с созданным ботом через @BotFather
- Доступ к Google Sheets с настроенной служебной учетной записью
- Разрешение на чтение/запись в Google Таблицу для служебной учетной записи
