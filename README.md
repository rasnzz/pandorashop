# PandoraShop - Telegram Bot для управления товарными запасами и рассылки

Этот бот сочетает в себе функции управления товарными запасами через Google Sheets и инструмент для массовой рассылки постов в Telegram каналы.

## Функционал

### Основной бот (Google Sheets)
- 🔍 Поиск по категории (Браслеты, Подвески, Серьги, Кольца, Наборы и пользовательские листы)
- 🔎 Поиск по артикулу
- 📝 Управление листами (создание, удаление, просмотр)
- 👁️ Просмотр карточек товаров с фото, названием, артикулом и остатками
- 🔄 Изменение количества товаров (+/-)
- 🗑️ Удаление товаров
- 🔄 Синхронизация с Google Sheets

### Спам-бот (Рассылка)
- 📢 Циклическая рассылка постов с заданными интервалами
- ⚙️ Управление через панель в основном боте
- ▶️ Возможность запуска/останова рассылки
- 📜 Просмотр логов и активности
- 📺 Настройка каналов для рассылки
- ✏️ Управление постами и параметрами рассылки

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/rasnzz/pandorashop.git
cd pandorashop
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте переменные окружения в файле `.env`:
```env
# Основной бот
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=список_id_администраторов_через_запятую
GOOGLE_SHEET_ID=id_вашего_документа_google_sheets
GOOGLE_CREDENTIALS_FILE=credentials.json

# Спам-бот (Telethon Userbot)
API_ID=ваш_api_id_из_telegram
API_HASH=ваш_api_hash_из_telegram
PHONE=ваш_телефон_с_плюсом_и_кодом_страны
```

4. Добавьте файл `credentials.json` с ключами доступа к Google API

5. Запустите бота:
```bash
python bot.py
```

## Использование

### Основной функционал
- `/start` - главное меню
- `/search_article` - поиск товара по артикулу
- `/create_sheet` - создание нового листа
- `/sync` - синхронизация с Google Sheets

### Спам-бот
- `/spam` или кнопка "📢 Спам-бот" в главном меню
- Панель управления рассылкой:
  - Запуск/остановка рассылки
  - Управление постами
  - Управление каналами
  - Настройка интервалов
  - Просмотр логов и статистики

#### Добавление постов
- Посты автоматически определяются по файлам в папке `spam_bot/texts/`
- Формат: `название.txt` - текст поста
- Для добавления фото: `название.jpg` (или .jpeg, .png) в папке `spam_bot/photos/`
- Имя файла фото должно совпадать с именем файла текста

#### Настройка каналов
- Добавление каналов через интерфейс бота
- Поддержка публичных каналов (@channel_name)
- Поддержка приватных каналов (-1001234567890)

## Настройка параметров рассылки

Через панель управления можно настроить:
- Интервал между постами (по умолчанию 120 секунд)
- Интервал между циклами (по умолчанию 3600 секунд)
- Список каналов для рассылки
- Тексты и фото постов

## Структура Google Таблицы

### Стандартные листы:

#### Браслеты:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул (уникальный идентификатор)
- Колонка D: Количество для размера 16 (13-14см)
- Колонка E: Количество для размера 18 (15-17см)
- Колонка F: Количество для размера 20

#### Подвески, Серьги, Наборы:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул
- Колонка D: Количество (единый размер)

#### Кольца:
- Колонка A: URL фото изделия
- Колонка B: Название изделия
- Колонка C: Артикул
- Колонка D: Количество для размера 50
- Колонка E: Количество для размера 52
- Колонка F: Количество для размера 54

## Папки проекта

- `spam_bot/` - файлы спам-бота
  - `texts/` - тексты постов
  - `photos/` - изображения к постам
  - `session/` - файл сессии Telethon
  - `logs/` - логи спам-бота
- `session/` - файл сессии основного бота
- `logs/` - логи основного бота

## Требования

- Python 3.10+
- Аккаунт Telegram Developer API (для спам-бота)
- Доступ к Google Sheets с настроенной служебной учетной записью
- Разрешение на чтение/запись в Google Таблицу для служебной учетной записи

## Безопасность

- Не храните файл .env в открытом репозитории
- Ограничьте доступ к боту только доверенным лицам
- Аутентификация через ADMIN_IDS
